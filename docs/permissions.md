# Permissions

This page lists all the permissions required to use the administrative features of **RMerchant**.

## Permission Nodes

| Permission Node | Description | Default |
| :--- | :--- | :--- |
| `rmerchant.admin` | Grants full access to all plugin commands (`reload`, `start`, `stop`, `addrecipe`, etc.). | OP |

::: info
Regular players do not need any permissions to interact with the Merchant NPC.
:::
